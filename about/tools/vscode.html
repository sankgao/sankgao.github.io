                    # VS Code 工具插件

                    # Chinese (Simplified)

                    Chinese (Simplified)open in new window 用于更改 VS Code 本地语言为中文界面。

                    # Makdown Preview Github Styling

                    Makdown Preview Github Stylingopen in new window 用于更改 VS Code 的内置 Markdown 预览以匹配 GitHub 的样式。

                    # Markdown Preview Enhanced

                    Markdown Preview Enhancedopen in new window 用于增强 Markdown 预览，提供许多有用的功能。例如：自动滚动同步、数学排版、mermaid、PlantUML、pandoc、PDF 导出、代码块、演示文稿编写器等。

                    快捷键功能
                    ctrl + k v打开侧面预览
                    ctrl + shift + v打开预览
                    Esc切换侧边栏目录

                    # markdown-formatter

                    markdown-formatteropen in new window 对用户提供了相对统一的格式，还提供了一些代码片段。

                    # background-cover

                    background-coveropen in new window 用于配置 Markdown 背景图。

                    • 点击底部切换背景图按钮 / Click the bottom toggle background button
                    • ctrl + shift + P > "backgroundCover - start"
                    • ctrl + shift + F7 > "Random update background and restart"
                    • VS Code 更新版本时会导致背景图消失，需要手动重新设置
