## 描述

`exit` 用于退出命令解释器或当前的批处理脚本。

## 选项

|  选项  |  描述  |
|  :----:  |  :----  |
|  `/b`  |  退出当前的批处理脚本，而不是退出 `Cmd.exe`。如果从批处理脚本外部执行，则退出 `Cmd.exe`  |
|  `/?`  |  在命令提示符下显示帮助  |

## 示例

例如：退出当前命令解释器。

```cmd
E:\test>cmd
Microsoft Windows [版本 10.0.19045.4651]
(c) Microsoft Corporation。保留所有权利。

E:\test>exit

E:\test>
```
