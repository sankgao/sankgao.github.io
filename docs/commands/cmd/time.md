## 描述

`time` 用于显示或设置系统时间。如果在不使用参数的情况下使用此命令，`time` 将显示当前系统时间，并提示输入新时间。

## 选项

|  选项  |  描述  |
|  :----:  |  :----  |
|  `/t`  |  显示当前时间，而不提示输入新时间  |
|  `/?`  |  在命令提示符下显示帮助  |

## 示例

例如：显示当前系统时间。

```cmd
E:\test>time /t
18:57

E:\test>
```

例如：将当前系统时间更改为 `5:30 PM`。

```cmd
E:\test>time 17:30:00
E:\test>time 5:30 pm
```
