## 描述

`ren` 或 `rename` 用于重命名目录或文件，但不能将重命名的目录或文件移动到其它路径下。

## 选项

|  选项  |  描述  |
|  :----:  |  :----  |
|  `/?`  |  在命令提示符下显示帮助  |

## 示例

### 重命名指定目录下的文件名

例如：将 `test01` 目录下的 `test01.txt` 文件，重命名为 `test02.txt`。

```cmd
E:\test>ren test01\test01.txt test02.txt

E:\test>
```

查看 `test01` 目录：

```cmd
E:\test>dir test01
```

### 重命名目录

例如：将当前目录下的 `test 03` 目录，重命名为 `test02` 目录。

```cmd
E:\test>ren "test 03" test02

E:\test>
```
