# HTML 颜色

颜色由红色、绿色、蓝色混合而成。

## 颜色值

颜色由一个十六进制符号来定义，这个符号由红色、绿色和蓝色的值组成（RGB）。

每种颜色的最小值是 0（十六进制：#00）。最大值是 255（十六进制：#FF）。

使用 HEX 值时需要在前面添加 `#` 号，如：`#F0F8FF`
使用 RGB 值时需要将数值放在 `rgb()` 中，如：`rgb(240,248,255)`
