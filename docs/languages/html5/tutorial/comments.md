注释标签 `<!-- 与 -->` 用于在 HTML 插入注释。

- 注释标签在开始标签中有一个 **惊叹号**，但是结束标签中 **没有** 惊叹号
- 浏览器不会显示注释，但是能够帮助记录您的 HTML 文档
- 您可以利用注释在 HTML 中放置 **通知** 和 **提醒信息**
- 可以注释一行文本，也可以注释多行文本

例如：

::: normal-demo 演示

```html
<!-- 这是一段注释 -->

<p>这是一个段落。</p>

<!-- 
  记得在
  此处
  添加信息
-->
```

:::
