C 语言除了用于计算机操作系统的编写，C 语言现在主要用于嵌入式系统的开发，汽车、照相机、家用电器等设备的底层系统都是用 C 语言编写的。

C 语言能够直接操作硬件、管理内存、跟操作系统交互，这使得它成为了一种非常接近底层的编程语言，非常适合有极高性能要求的程序。
